Project Overview

项目概况

🎮

游戏类型

休闲合成闯关游戏。玩家在网格棋盘上合并道具获取更高级健身器材，为胖鹅建造健身房，同时在关卡中完成目标挑战推进剧情。

🔄

核心循环

合成道具体验 → 获取高阶道具 → 为健身房供能 → 完成关卡目标 → 获得资源奖励 → 继续合成，双循环互相供能形成闭环。

💰

商业化模型

激励视频广告（体力双倍/道具福袋）+ 月卡 + 赛季通行证 + 皮肤内购，非付费用户完整游戏体验，付费仅用于加速。

📅

当前进度

2026.07 策划案 V1.0 完成，含完整玩法设计、数值体系、商业化方案、技术规格与运营规划，待开发落地。

Core Mechanics

核心玩法设计

合成系统 · 关卡设计 · 任务系统

🧩

合成系统 — Merge Mechanic

8×8 或 9×9 网格棋盘，通过拖拽或点击交换相邻道具，将 3 个或以上相同道具合并为更高级道具。合成公式遵循阶梯递进逻辑：

合成链 T1 基础 T2 T3 T4 高级 T5 稀有
🍳 食品链 鸡蛋煎蛋面包能量棒鸡胸肉套餐
🏋️ 器材链 哑铃杠铃跑步机划船机全套健身器
🧘 装饰链 瑜伽垫瑜伽球拳击手套运动水壶金牌奖杯
✨ 特殊链 肥鹅便便清洁刷肥皂毛巾健身教练卡
T1 白色 T2 绿色 T3 蓝色 T4 紫色 T5 橙色

合成规则：最低 3 个相同道具相邻即可合成（含斜向不相邻）；合成高阶道具后有概率掉落能量气泡，用于激活特殊技能；棋盘满格且无可合成项时游戏失败。

🎯

关卡设计 — Level System

每个关卡在限定步数内完成特定消除目标，同时棋盘上方显示倒计时与进度条。关卡类型分为四类：

🎯 收集型
在限定步数内收集指定数量的特定道具
例：消除 25 个煎蛋
❄️ 消除型
消除所有目标区域或带有标记的格子
例：清除所有冰块格子
⚡ 组合型
同时包含收集目标与障碍物清除
例：收集15个能量棒+清除所有箱子
🦴 Boss 型
消除大型障碍物（胖鹅堆积的脂肪方块）
例：3步内消除所有脂肪方块
阶段关卡范围步数目标复杂度特征
教程期1–30 关15–20 步单一目标引导清晰，步数充足
成长爬坡31–100 关12–15 步引入障碍物步数递减，需要策略
策略期101–200 关10–12 步多目标并行需要规划合成顺序
挑战期201+ 关8–10 步随机事件+加成高难度，技巧性强
📋

任务系统 — Quest System

三大任务体系驱动玩家持续活跃：

📅 每日任务
每天重置：完成3关、合成5次T3道具体验等，奖励金币与体力
🏆 成就任务
一次性达成，奖励永久头像框、背景、称号等
⚔️ 赛季任务
配合赛季活动，奖励限定装扮（春节舞狮/万圣节南瓜等）
Game Systems

配套游戏系统

装扮系统 · 剧情系统 · 社交系统

👕

装备/装扮系统

玩家使用收集的道具为胖鹅解锁健身房装备（跑步机/拳击台/杠铃架/饮水区/休息沙发），解锁后胖鹅有专属健身动画。赛季限定皮肤增加商业化深度。

📖

剧情系统

每通关 20 关解锁一个剧情章节，以胖鹅日记漫画形式呈现，讲述减肥历程趣事。全程配音（每章节约 30 秒语音），配合 Q 版动画提供情感共鸣。

👥

微信社交系统

好友排行榜 / 体力赠送（每日 10 次上限）/ 邀请助力分享 / 微信状态同步。轻社交设计，避免强制绑定，降低用户流失风险。

Monetization

商业化设计

💰

付费点与广告设计

采用「免费增值 + 轻量内购」模式，严格避免 pay-to-win，核心内容均可通过时间投入解锁：

付费类型内容描述单价频次
体力购买立即补充 5 点体力¥6日常
道具礼包随机 T3–T5 道具福袋¥6 / ¥30日常
首充礼包6元首充得60钻+限定皮肤¥6一次性
月卡每日30体力+100金币¥30/月月订阅
赛季通行证赛季专属道具+限定胖鹅皮肤¥68/赛季赛季
「激励视频广告（体力双倍/道具福袋/步数加成）为核心收入来源，非强制，用户主动选择，整体设计保证免费用户体验完整。」

竞品分析报告 — 核心结论

🎯 市场空白

⏰ 窗口期
大厂随时可入局复制，6 个月内可能出现模仿者，窗口期是生死线
💡 战略建议
上线前快速建立 IP 认知，上线后精细化女性用户运营，内容护城河持续深化
