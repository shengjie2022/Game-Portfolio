一、产品定位

1.1 初始定位（1.0-2.0版本）

《原神》上线初期以"开放世界探索驱动"为核心卖点：

  • 目标用户：主机/PC端开放世界游戏玩家（《塞尔达传说》受众）
  • 核心体验：大地图探索、解谜、元素交互、世界任务叙事
  • 差异化：跨平台（PC/手机/主机）+ 免费游玩 + 持续内容更新
  • 留存逻辑：探索新鲜感 → 地图100%收集成就感

1.2 转型定位（3.0-5.0版本）

逐步转向"角色养成+剧情驱动"：

  • 目标用户扩展：移动端泛二次元用户、剧情向玩家
  • 核心体验转移：角色抽取与培养 → 主线/支线剧情沉浸 → 周期性深渊挑战
  • 底层逻辑变化：从"内容消耗型"（探索完即流失）转向"角色资产积累型"（持续投入养成）

1.3 转型动因分析

维度 开放世界驱动的问题 角色养成驱动的优势
内容产能 大地图开发成本高、周期长（6个月/国） 角色产出周期短（6周/角色），可持续
付费转化 探索内容免费，付费点模糊 角色抽取直接关联付费，转化路径清晰
用户留存 探索完成后进入"长草期"，流失率高 角色养成是长期过程，留存曲线更平缓
社区生态 攻略内容同质化 角色二创、厨力讨论、同人内容爆发

二、核心循环

2.1 初始核心循环（1.0-2.0）

探索地图
→
发现解谜
→
获取宝箱/神瞳
→
提升体力上限
→
探索更远区域

问题：循环终点是"地图100%"，完成后缺乏持续目标。

2.2 转型后核心循环（3.0+）

版本更新
→
新剧情/新角色UP
→
抽卡获取角色
→
养成(材料/圣遗物)
→
深渊验证强度
  • 剧情成为角色"售前介绍"：角色PV + 传说任务 → 建立情感连接 → 驱动抽取意愿
  • 深渊成为"养成终点验证"：周期性重置 + 环境BUFF → 刺激新角色需求
  • 圣遗物系统：随机词条提供"无限养成深度"，延长角色培养周期

2.3 双循环架构

循环类型 目标 周期 付费关联
外循环：内容消费 剧情体验、地图探索 6周/版本 低（免费内容）
内循环：角色养成 角色抽取、培养、验证 每日/每周 高（抽卡付费）
核心洞察

转型本质：将外循环的"内容新鲜感"转化为内循环的"角色资产积累"。

三、商业化设计

3.1 付费结构演变

版本阶段 主要付费点 设计逻辑
1.0-2.0月卡 + 首充 + 角色池探索期用户付费意愿低，依赖"性价比"诱导
3.0-4.0角色池（双UP）+ 武器池 + 皮肤角色成为核心资产，剧情驱动抽取决策
5.0+角色池 + 武器池 + 皮肤 + 礼包矩阵成熟付费体系，分层收割不同付费能力用户

3.2 "深渊"作为商业化杠杆

机制设计：每15天重置，每期配置"环境BUFF"（特定元素/角色伤害加成）

  • 环境BUFF → 暗示"当期UP角色适配"：刺激抽取
  • 满星压力 → 强度焦虑：驱动命座/武器抽取
  • 免费原石投放 → "差一点就能抽到"：制造心理落差

四、可迁移洞察

4.1 对独立游戏《蟹化进化2》的启示

《原神》策略迁移至独立游戏
剧情驱动角色价值用"蟹化值"叙事包装进化分支，让每个选择有情感重量
周期性内容更新DLC矩阵规划（新生态群落/新结局），保持用户期待
社区二创生态提供世界观设定集/角色原画，激励UGC内容

方法论洞察

内容型产品的"双循环"设计：免费外循环拉新，付费内循环留存变现。

方法论洞察

方法论洞察

社区即产品：UGC内容不仅是营销，更是产品体验的延伸。

抽卡沉船：兹白歪了的那天

说实话，我氪过大月卡小月卡，但真正让我"心死"的一次，是兹白池子里歪了的那一刻。不是因为损失了多少钱，而是那种"我明明已经很努力在攒、在规划了，结果还是没得到"的无力感。

但冷静下来想想，这种挫败感本身，也是抽卡设计的一部分。歪了才会更珍惜下一次不歪的时候——原神用概率的不确定性，把"抽到"这件事的情绪价值拉满了。情绪价值本身是可以明码标价的，这话听起来功利，但我的实际体验告诉我确实如此。

策划视角：如果是我在设计抽卡系统，我会更关注"沉船后的情绪修复"机制。比如歪了之后给一个小的进度积累提示（"下次保底不歪的概率提升至80%"），让玩家觉得"虽然这次输了，但下一次我离赢更近了"，而不是一棍子打死。

艾尔海森：理性的人为什么会让我觉得真实

他是我最有感触的角色，不是因为他有多强，而是因为他的故事里有一种很稀缺的价值观：他不是没有能力改变世界，而是清楚自己愿意为改变付出多少代价。

传说任务里，他当代理贤者是因为"不想让更糟糕的人来做这件事"，但干完该干的事，他立刻就辞了。他不是不在乎——他只是不愿意为了"看起来很厉害"而付出超过他认为值得的代价。他和室友卡维吵架，本质上是两种价值观的碰撞：卡维觉得有才华的人应该去改变更多人，艾尔海森觉得管好自己已经够累了。

艾尔海森最后选择辞职，不是因为他不在乎，而是因为他太清楚自己愿意为改变付出多少代价。这种理性又自私的选择，反而比"为了拯救世界牺牲一切"更打动我。
策划视角：反差感角色最难设计的地方在于"度"——太冷会让人觉得冷漠、太暖又破坏了人设。艾尔海森的处理方式是：他的照顾永远是通过第三方体现的，而不是直接说出口。这种"不说但做了"比直接表达更有效。

让我真正佩服的细节：配件背后的性格语言

很多人说原神的角色设计好看，但我觉得真正厉害的不只是好不好看，而是你能不能从角色身上的每一个物件读出这个人的故事。

艾尔海森胸前挂着的那个绿色书签，是须弥知识之神的象征，对应他书记官的身份和学者气质；迪希雅耳环上那朵沙漠之花，代表她来自沙漠佣兵团的出身；荒泷一斗系着的"荒泷派"头巾，就是他这个帮派大哥的性格证明。

如果我在设计角色配件

我会问自己：玩家第一次注意到这个配件时，能不能猜到它代表什么？猜对了，说明它有记忆点；猜错了但看完解释觉得"合理"，说明它有深度。最糟糕的是玩家根本注意不到它——那它就只是贴图，不是设计。

圣遗物改动：官方做了，但我有别的想法

官方确实在优化圣遗物体验：5.5版本的祝圣之霜能让强化的副词条至少命中2条，6.0开始还能在强化前看到第4词条是什么。这些改动在减少"纯看脸"的比例，方向是对的。

但说实话，这些改动没有解决我的核心问题。我的痛苦不是"不知道会不会歪"，而是"我已经刷了三百次秘境了，我的角色到底有没有变强"——这个进展感，太模糊了。

如果是我来设计圣遗物：与其继续优化"强化的确定性"，不如做"强化的反馈感"——比如强化有视觉特效、每10级解锁一个小型角色动作变化、或者有专门的"养成记录"页面让你看到自己角色从0到毕业的全路径。这样玩家看到的不只是"还差多少"，还有"已经走了多远"。

从玩家到策划：我在原神里学到了什么

  • 角色即内容：每个角色附带专属剧情和PV，形成"追番式"期待，这是比任何广告都强的付费驱动
  • 弱压力环境的价值：无PVP、无排行榜，让社恐玩家也能长期留存，这部分用户的付费转化往往更高（因为没有竞争焦虑，只有纯粹的喜欢）
  • 反馈设计的优先级：确定性不如反馈感重要，玩家需要知道自己走了多远，而不只是知道终点在哪里
  • 配件是性格语言：每个物件都应该能让人读出故事，否则它就只是贴图
